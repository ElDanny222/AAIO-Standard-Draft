# Contributing to AAIO Standard Draft

Thank you for your interest in contributing to the Agentic AI Optimization (AAIO) Standard Draft.

## How to Contribute

### Reporting Issues

- Use GitHub Issues to report bugs or suggest improvements.
- Include a clear description, relevant context, and steps to reproduce (if applicable).

### Proposing Changes

1. **Fork** the repository.
2. **Create a branch** from `main` (e.g., `feature/your-change`).
3. **Make your changes** following the guidelines below.
4. **Submit a Pull Request** with a clear description of the change.

### Writing Guidelines

- **Language**: Primary language is English. Finnish translations are welcome and encouraged.
- **Format**: Follow IETF/W3C-style specification formatting.
- **Versioning**: All specifications use semantic versioning (e.g., `v0.1-draft`).
- **Status labels**: Use `Draft`, `RFC`, or `Stable` to indicate document maturity.

### Specification Contributions

- Specifications go in `specs/`.
- JSON Schema definitions go in `schemas/`.
- Reference implementations go in `examples/`.
- Supporting documentation goes in `docs/`.

### Code of Conduct

Be respectful, constructive, and professional in all interactions.
